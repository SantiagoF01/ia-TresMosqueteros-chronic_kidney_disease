# ia-TresMosqueteros-chronic_kidney_disease
Este repositorio corresponde al proyecto semestral de la asignatura Inteligencia Artificial 1 grupo C2, desarrollado por Santiago Ardila, Santiago Flórez y María Lucía Rodríguez. En breves palabras, el proyecto consiste en el estudio de la enfermedad renal crónica y la evaluación de diferentes modelos que sean de utilidad para este problema.
